\# Clothing Classification using CNN (StyleNest)



\## Overview

This project uses a Convolutional Neural Network (CNN) to classify clothing images from the Fashion MNIST dataset.



\## Problem Statement

Manual tagging of clothing products is time-consuming and error-prone. This project automates classification to improve efficiency and accuracy.



\## Features

\- CNN-based image classification

\- 10 clothing categories

\- Model evaluation using accuracy, precision, recall, F1-score

\- Confusion matrix analysis



\## Tech Stack

\- Python

\- TensorFlow / Keras

\- NumPy

\- Scikit-learn

\- Matplotlib



\## Dataset

Fashion MNIST dataset:

\- 60,000 training images

\- 10,000 test images

\- 28x28 grayscale images



\## Results

\- Achieved \~90% accuracy



\## Use Case

Can be used in e-commerce platforms to automate product tagging and improve search \& recommendations.
